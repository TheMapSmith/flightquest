# flightquest
What's that on the ground?

# Installation
- Make sure [Node.js](https://nodejs.org) is installed
- Clone or download the repository 
- Open Terminal app (OSX) or PowerShell (WIN)
- Change directory (`cd`) to the app directory
- run `npm install`

# Start
`npm start`
